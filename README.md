# 🎮 Centro de Juegos con IA

Aplicación en Python que reúne múltiples juegos y herramientas en un solo programa, incluyendo inteligencia artificial básica que aprende del jugador.

---

## 🚀 Funcionalidades

- 🎮 Piedra, Papel o Tijera con IA inteligente
- 🎯 Juego de adivinanza con intentos
- 🧮 Calculadora básica
- 📊 Sistema de estadísticas (victorias, derrotas, empates)
- 💾 Guardado automático de datos (JSON)

---

## 🧠 Inteligencia Artificial

La IA analiza tus jugadas anteriores y adapta su estrategia para intentar ganarte.

- Aprende de tus elecciones
- Calcula probabilidades
- Toma decisiones basadas en tu comportamiento

---

## ▶️ Cómo ejecutar

1. Descargar el repositorio
2. Abrir una terminal en la carpeta
3. Ejecutar:

```bash
python centro_juegos_ia_pro.py
```

---

## 📌 Requisitos
